### 函示庫連結 -lm

```
// gcc erf.c -o erf -lm 連結math.h
#include <math.h> //erf誤差函數, sqrt
#include <stdio.h>
int main() {
  printf("normal(0,1) distribution between(-1.96, 1.96) is %g\n", erf(1.96*sqrt(1/2.)));
}
```

### linux指令: vim

```
i: 進入編輯模式
Esc: 離開編輯模式
o: 插入一行
yy: 複製一行
p: 貼上一行
dd: 刪除一行
cc: 剪下一行
wq: 儲存並離開
```
